# [Training Sets]

## Description
Automates the process of training your Digimon on the Digifarm by using each of the "Training Set C" items to bring each stat up to 9999. This will require about 4 million yen PER STAT!

## Game Version
- Game: [Digimon Story: Time Stranger]
- Version: [1.000.004]
- Region: [US]
